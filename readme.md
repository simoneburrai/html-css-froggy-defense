# CSS Flex
Il flexbox, o flexible box, è un modello che permette di creare layout, anche complessi, in maniera pulita, rapida e responsive.

Qui di seguito potete trovare informazioni più dettagliate riguardo il **flex-box**:

[CSS Flex](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)
